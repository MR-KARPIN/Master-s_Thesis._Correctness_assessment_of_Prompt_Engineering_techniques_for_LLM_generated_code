Here's a simple function that uses the `in` keyword to check if an element is in a list:

```python
def search_element(lst, element):
    if element in lst:
        return True
    else:
        return False
```

You can use it like this:

```python
my_list = [1, 2, 3, 4, 5]
print(search_element(my_list, 3))  # prints: True
print(search_element(my_list, 6))  # prints: False
```

If you also want to return the index of the element if it is found, you can modify the function like this:

```python
def search_element(lst, element):
    if element in lst:
        return lst.index(element)
    else:
        return -1
```

In this version, the function returns the index of the element if it is found, and -1 if it is not found.
